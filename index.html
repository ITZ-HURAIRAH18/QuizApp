Our Features

Dynamic Question

Users can upload an Excel file containing MCQs. The app reads and displays the selected number of questions dynamically.

Read More
Real-Time Answer Checking

Users select an answer, and it instantly highlights correct (✅ green) or wrong (❌ red). The correct answer is always shown if a wrong option is chosen.

Read More
Countdown Timer

Each question has a countdown timer, forcing users to answer within time. If time runs out, the app automatically highlights the correct answer and moves to the next question.
